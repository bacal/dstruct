# Dstruct - A data structure library written in C

### Building
There are no external dependencies for this library just build with a build system of your choice.

### GNU Make
```
$ make
```
- Optionally to make the demo program
```
$ make demo
```

### CMake (All Platforms)
```
$ mkdir build
$ cd build
$ cmake ..
$ make
```

### Meson
```
$ meson build
$ cd build
$ meson compile
```
